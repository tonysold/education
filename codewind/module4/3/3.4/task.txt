Ваша задача - создать функцию changeFontSize, которая будет принимать два параметра: selector и fontSize. 
Функция должна найти все элементы на странице,
соответствующие заданному CSS-селектору selector.
Затем для каждого найденного элемента функция должна изменить размер шрифта на заданный размер fontSize.

Пример вызова функции:
changeFontSize('.myElement', '20px');

Подсказка:
Для изменения размера шрифта элементов вы можете использовать свойство style.fontSize элемента
и присвоить ему значение заданного размера.

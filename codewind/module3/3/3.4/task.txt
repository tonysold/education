Развернуть и запустить готовое веб-приложение, созданное с помощью приемов ООП. Для этого выполнить следующие действия:

1) Скачать архив проекта: (файл project)
2)  Содержимое архива поместить с папку хоста Open Server (например, localhost); 
3) С помощью PhpMyAdmin создать новую базу данных с именем oopscrud; 
4)     Импортировать дамп базы данных oopscrud.sql; 
5)     Запустить веб-приложение http://localhost
